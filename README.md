## Quick Reference

### Time-Critical SAR Operations

When 24-hour ATC notification is NOT possible:
- Use **STSC-004 parameters only**
- Maximum altitude: **400' AGL**
- BVLOS permitted
- No ATC notification required
- Terrain-following search patterns

### Descend Protocol

```
1. IDENTIFY threat
2. CALL "DESCEND, DESCEND, DESCEND"
3. SWITCH to Manual Control
4. MANEUVER downward to safe altitude
```
